Temperature Project

You are tasked with the following:

  • Create a function that will run when the page fully loads that askes the user what temperature it currently is and what temperature the thermostat should be programmed to. The function must check whether the user enters a valid number (between 45° and 95°). The function should also store the values entered in the current temerapture space and programmed temperature space.
  • Create two functions, called incTemp and decTemp. These functions will be used to manually override the programmed temperature when the red-up arrow is clicked and when the blue-down arrow is clicked. These should store the new temperature in the override temperature space and should also check whether the thermostat should be heating or cooling to reach the desired temperature. They should then change the text inside the display to say 'Heating' or 'Cooling'
  • Explain what other features we could add to our thermostat that might make it more useful. Explain what functions might be needed to accomplish these and how you might go about programming this. Note: You do not have to actually program it, but you should still be using programming words like "if/else statements", "function parameters", and other words we have been learning.
